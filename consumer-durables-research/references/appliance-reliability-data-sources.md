# Appliance Reliability Data Sources

## Primary: Real Service Call Data

- **Yale Appliance Blog** — `blog.yaleappliance.com`
  - Annual "Best Front-Load Washers" post with brand-level service rates
  - 2026 data: LG 2.7%, GE Appliances 8.5%, GE Profile 10.5%, category avg 4.6%
  - Also tracks top-loaders, dryers, dishwashers
  - Yale stopped selling Electrolux ~5 years ago (service support concerns)

## Secondary: Lab Testing + Predicted Reliability

- **Consumer Reports** — `consumerreports.org`
  - "Which Brands Make the Most Reliable Washing Machines" (annual)
  - Based on member surveys, not actual repair calls
  - Typically ranks LG, Speed Queen, Miele, Bosch as most reliable

## Owner Forums

- **Reddit r/Appliances** — owner experiences, purchase advice
- **Reddit r/appliancerepair** — repair tech insights, failure mode patterns
- Key signal: if multiple owners report the SAME failure in a short time window, it's a design defect

## Retailer Review Aggregates

- **Home Depot** — high volume, filter by 1-star for failure patterns
- **AJ Madison** — detailed owner reviews, often includes model-specific issues
- **Lowe's** — similar to Home Depot
- **Abt** — higher-end customer base, usually better detail

## Repair Tech YouTube Channels

- Ben's Appliances and Junk — common failure modes, diagnostic tips
- Yale Appliance — buying guides with reliability data

## Recalls / Legal

- Search pattern: `[brand] [product type] recall class action [year]`
- **Class action settlements to know:**
  - Electrolux mold settlement (older front-loaders, pre-2020)
  - LG mold settlement (older front-loaders)
  - Whirlpool pressure sensor / control board extended coverage
